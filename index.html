    JavaScript is a widely used scripting language to make web pages interactive and develop web applications. The script is transferred to the web browser using HTML and is executed there. jQuery is a free JavaScript library for dynamic and interactive websites, including for editing the DOM and CSS and interaction with the web server.

    Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

    Wireframing defines the elements that need to be present from page to page.
